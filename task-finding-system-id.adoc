= 查找连接器的系统 ID
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
为了帮助您入门， NetApp 代表可能会要求您提供 Connector 的系统 ID 。此 ID 通常用于许可和故障排除目的。

.步骤
. 在 Cloud Manager 控制台右上角，单击帮助图标。
. 单击 * 支持 > 连接器 * 。
+
系统 ID 显示在顶部。
