йҖҡиҝҮдёәBlueXPжҺ§еҲ¶еҸ°еҮҶеӨҮзҪ‘з»ңиҝһжҺҘгҖҒжіЁеҶҢе’ҢеҲӣе»әеёҗжҲ·гҖҒ(еҸҜйҖү)еҲӣе»әConnectorд»ҘеҸҠи®ўйҳ…BlueXPгҖҒеңЁж ҮеҮҶжЁЎејҸдёӢејҖе§ӢдҪҝз”ЁBlueXPгҖӮ

еңЁж ҮеҮҶжЁЎејҸдёӢгҖҒз”ЁжҲ·еҸҜд»ҺеҹәдәҺWebзҡ„жҺ§еҲ¶еҸ°д»Ҙдә‘жңҚеҠЎзҡ„еҪўејҸи®ҝй—®BlueXPгҖӮејҖе§Ӣд№ӢеүҚгҖҒжӮЁеә”дәҶи§Ј "BlueXPеёҗжҲ·"пјҢ "иҝһжҺҘеҷЁ"пјҢе’Ң "йғЁзҪІжЁЎејҸ"гҖӮ

"еҮҶеӨҮзҪ‘з»ңд»ҘдҪҝз”ЁBlueXPжҺ§еҲ¶еҸ°"

и®ҝй—®BlueXPжҺ§еҲ¶еҸ°зҡ„и®Ўз®—жңәеә”иҝһжҺҘеҲ°зү№е®ҡз«ҜзӮ№д»Ҙе®ҢжҲҗдёҖдәӣз®ЎзҗҶд»»еҠЎгҖӮеҰӮжһңжӮЁзҡ„зҪ‘з»ңйҷҗеҲ¶еҮәз«ҷи®ҝй—®гҖҒеҲҷеә”зЎ®дҝқе…Ғи®ёиҝҷдәӣз«ҜзӮ№гҖӮ

"жіЁеҶҢе№¶еҲӣе»әеёҗжҲ·"

иҪ¬иҮі "BlueXPжҺ§еҲ¶еҸ°" е№¶жіЁеҶҢгҖӮжӮЁеҸҜд»ҘйҖүжӢ©еҲӣе»әеёҗжҲ·гҖҒдҪҶеҰӮжһңжӮЁеҸ—йӮҖеҠ е…ҘзҺ°жңүеёҗжҲ·гҖҒеҲҷеҸҜд»Ҙи·іиҝҮиҜҘжӯҘйӘӨгҖӮ

жӯӨж—¶гҖҒжӮЁе·Ізҷ»еҪ•е№¶еҸҜд»ҘејҖе§ӢдҪҝз”ЁеӨҡз§ҚBlueXPжңҚеҠЎгҖҒдҫӢеҰӮDigital AdvisorгҖҒAmazon FSX for ONTAP гҖҒAzure NetApp Files зӯүгҖӮ "дәҶи§ЈжІЎжңүConnectorеҸҜд»Ҙжү§иЎҢзҡ„ж“ҚдҪң"гҖӮ

еҲӣе»әиҝһжҺҘеҷЁ

жӮЁж— йңҖдҪҝз”ЁConnectorеҚіеҸҜејҖе§ӢдҪҝз”ЁBlueXPгҖҒдҪҶжӮЁеҸҜд»ҘеҲӣе»әдёҖдёӘConnectorжқҘи§Јй”ҒжүҖжңүBlueXPеҠҹиғҪе’ҢжңҚеҠЎгҖӮConnectorжҳҜдёҖж¬ҫNetAppиҪҜд»¶гҖҒеҸҜи®©BlueXPз®ЎзҗҶж··еҗҲдә‘зҺҜеўғдёӯзҡ„иө„жәҗе’ҢжөҒзЁӢгҖӮ

BlueXPеёҗжҲ·з®ЎзҗҶе‘ҳеҸҜд»ҘеңЁдә‘жҲ–еҶ…йғЁзҪ‘з»ңдёӯеҲӣе»әConnectorгҖӮ
